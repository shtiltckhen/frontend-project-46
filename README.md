### genDiff
Compares two formatted .json or .yml(.yaml) files and output the result as 'stylish', 'plain' or 'json'.

### Requirements:
* npm version 10.2.5
* node version 21.1.0

### Installing:
`git clone git@github.com:shtiltckhen/frontend-project-46.git`  
`make install`
